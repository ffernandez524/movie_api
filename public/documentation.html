Cinenotes

This API provides users with a way to easily download information about your favorite Movies and Directors! Users will be able to create an account to save their favorites and add some info about themselves!

Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
Get a list of Movies /movies GET None JSON Object holding a list of all movie titles
Get data of a single movie by title /movies/[title] GET None JSON Object holding a data about a single movie. Ex: { title: "The Avengers", genre: "Action", director: "Joss Whedon", imageURL: "/img/the_avengers.jpg", featured: "yes" }
Get description of a genre by name /genres/[Name] GET None JSON Object holding a description about a genre. Ex: { name: "Action", description: "Movies in the action genre contain fast-paced scenes depicting exciting events.", }
Get data of a director by name /directors/[Name] GET None JSON Object holding a description about a director. Create new user /users POST JSON object holding data about the user to add. Ex: { username: "spicyJalapeno99" name: "Bob Jones", email: "movieluvr@gmail.com", bio: "I'm a huge fan of thrillers!", dob: "7/21/1990" } JSON object holding data about the newly created user. Ex: { id: 2, username: "spicyJalapeno99", favorites: "Lord of the Rings, Battlestar Galactica", name: "Bob Jones", email: "movieluvr@gmail.com", bio: "I'm a huge fan of thrillers!", dob: "7/21/1990" }
Update User Info /users/[id] PUT None Text message indicating updated username
Add Favorite Movies /users/[username]/favorites/[movie] PUT None Text message indicating the movie was added
Remove Favorite Movies /users/[username]/favorites/[movie] Delete None Text message indicating the movie that was removed
Remove User /users/[id] Delete None Text message indicating the user was removed
