Cinenotes

This API provides users with a way to easily download information about your favorite Movies and Directors! Users will be able to create an account to save their favorites and add some info about themselves!

Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
GET List of Movies /movies GET None JSON Object holding a list of all movies
GET Data of a single movie by title /movies/[title] GET None JSON Object holding a data about a single movie. Ex: { Title: "The Avengers", Description: "Earth's mightiest heroes must come together and learn to fight as a team if they are going to stop the mischievous Loki and his alien army from enslaving humanity." Genre: { Name: 'Action', Description: 'Movies in the action genre contain fast-paced scenes depicting exciting events.' } Director: { Name: 'Joss Whedon', Bio: 'Joss Whedon is an American filmmaker, composer, and comic book writer.', Birth: '1964', Death: null } ImagePath: 'theavengers.png', Featured: true }
Get the description of a genre by name /genres/[Name] GET None JSON Object holding a description about a genre. Ex: { name: "Action", description: "Movies in the action genre contain fast-paced scenes depicting exciting events.", }
Get data of a director by name /directors/[Name] GET None JSON Object holding a description about a director. Ex: { Name: 'Joss Whedon', Bio: 'Joss Whedon is an American filmmaker, composer, and comic book writer.', Birth: '1964', Death: null }
Create new user /users POST JSON object holding data about the user to add. Ex: { Username: "spicyJalapeno99" Password: "ASdji3j29uaS", Email: "movieluvr@gmail.com", Birthday: "12-11-1923", } JSON object holding data about the newly created user. Ex: { _id: 23, Username: "spicyJalapeno99" Password: "ASdji3j29uaS", Email: "movieluvr@gmail.com", Birthday: "12-11-1923", }
Update User Info /users/[Username] PUT JSON object holding new user info Ex: { Username: "spicyJalapeno91" Password: "AD2SAD2", Email: "movieluvr24@gmail.com", Birthday: "12-11-1923", } JSON object showing updated user info. Ex: { _id: 23, Username: "spicyJalapeno91" Password: "AD2SAD2", Email: "movieluvr24@gmail.com", Birthday: "12-11-1923", Favorites: [] }
Add Favorite Movies /users/[Username]/favorites/[MovieID] POST None JSON Object containing user info including added favorites Ex: { _id: "63ffbe45ed030b1d0a1bfe3e", Username": "woolyemu21", "Password": "cacaw24!", "Email": "toowooly252@gmail.com", "Birthday": "1977-05-22T00:00:00.000Z", "Favorites": [ "63ffb7e8ed030b1d0a1bfe32", "63ffb93eed030b1d0a1bfe34" ] }
Remove Favorite Movies /users/[username]/favorites/[movie] Delete None JSON Object containing updated user info. EX: { "_id": "63ffbe45ed030b1d0a1bfe3e", "Username": "woolyemu21", "Password": "cacaw24!", "Email": "toowooly252@gmail.com", "Birthday": "1977-05-22T00:00:00.000Z", "Favorites": [ "63ffb7e8ed030b1d0a1bfe32", "63ffb93eed030b1d0a1bfe34" ], "FavoriteMovies": [ "63ffbc8fed030b1d0a1bfe3a" ] }
Remove User /users/[id] Delete None Text message indicating the user was removed
